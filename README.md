Önce discord mesaj çubuğunun koordinatlarını bulmak için koordinat.py dosyasını terminalde başlatın.
Ardından discord mesaj çubuğuna tıklayın.
Çıkan Koordinatları sil.py dosyasındaki sırasıyla x, y yazan yere yazın.
Ardından terminal üzerinden sil.py dosyasını çalıştırın.
Kodun kaç kere tekrar etmesini istediğinizi count değişkenine yazın. (Varsayılan = 50)
